# 모바일 스크린샷 PDF 생성기

휴대폰에 있는 이미지를 업로드하여 여러 장을 하나의 PDF로 변환하고 공유할 수 있는 웹 도구입니다.
서버 없이 브라우저에서만 동작합니다.

---

## 주요 기능

### 이미지 업로드

* 휴대폰 갤러리에서 다수 이미지 선택 가능
* 선택 순서 기준으로 PDF 생성

### 이미지 편집 (Crop)

* 이미지 선택 시 편집 화면 진입
* 드래그로 영역 지정
* 핀치 줌, 이동, 리사이즈 지원

### PDF 생성

* A4 기준 자동 레이아웃
* 한 줄 3개 배치
* 여러 페이지 자동 생성

### PDF 미리보기

* 생성된 PDF를 화면에서 바로 확인 가능

### 공유 기능

* iOS: 카카오톡, 메일 등으로 파일 첨부 공유 가능
* Android: 다운로드 후 파일 앱에서 공유

---

## 사용 방법

1. 웹페이지 접속
2. 이미지 선택 (복수 선택 가능)
3. 필요 시 이미지 클릭 후 편집
4. PDF 생성 버튼 클릭
5. 공유 버튼 클릭 또는 다운로드 후 사용

---

## 플랫폼별 동작 차이

### iOS (Safari)

* 파일 공유 기능 정상 동작
* 메신저 및 메일로 첨부 전송 가능

### Android (Chrome)

* 파일 공유 기능 일부 제한
* 다운로드 후 파일 앱을 통해 공유 필요

이 차이는 브라우저의 Web Share API 지원 범위에 따른 것입니다.
